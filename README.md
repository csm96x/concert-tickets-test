## Concert Ticket Booking Automation
### This project contains an automation test for a simple concert ticket booking website, made with HTML.

## Technologies used: 
- Python
- Pytest
- Selenium

## Test scenarios: 
- Add a valid number of tickets
- Add zero tickets
- Add empty input

## How to run:
1. install requirements:
- pip install pytest
- pip install selenium

2. run the tests:
 pytest test_ticketing.py
